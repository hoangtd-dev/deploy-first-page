Text center

css property value description
text-align center You can make text in center with text-align: center;
margin 0 auto You can make text in center with margin: 0 auto;. Css will auto alignment
display flex

You can make text in center with display flex and combine some property

display: flex;. It will make element in flex mode

justify-content: center;. It will align content to center horizontally

align-items: center;. It will align content to center vertically
